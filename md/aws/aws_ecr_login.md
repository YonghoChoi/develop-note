* pip 설치

  ```shell
  $ sudo apt-get install python-pip python-dev build-essential
  ```

* awscli 설치

  ```shell
  $ pip install --upgrade --user awscli
  ```

* aws configuration 설정

  ```shell
  $ aws configure
  ```

  * ACCESS KEY와 SECRET KEY 입력

* aws ecr로 로그인

  ```shell
  $ aws ecr get-login --region ap-northeast-1
  ```
